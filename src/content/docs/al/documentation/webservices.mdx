admite tres tipos de servicios web: API (REST), SOAP y OData. Los servicios web son una forma liviana y estándar de la industria de hacer que la funcionalidad de la aplicación esté disponible para varios sistemas y usuarios externos. Puede crear y publicar funcionalidades como servicios web de API REST para leer o escribir datos en Business Central.

## Servicios web API REST

La pila de API de Business Central está optimizada para el rendimiento y es la forma preferida de integrar Business Central con sistemas externos.

## Servicios web OData

Con los servicios web OData, puede publicar páginas, unidades de código y consultas como puntos finales de servicio web ODataV4.

 Los servicios web OData exponen un documento XML de modelo de datos de entidad (EDMX) que contiene metadatos para todos los servicios web publicados. La implementación de OData en Business Central también admite el formato JSON, un formato menos detallado que puede funcionar mejor en entornos con poco ancho de banda.
